co::thread class
#include <co/thread.hpp>

represents a task, running concurrently in the event loop

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

can be in two stated: to be joined or detached. Usage:

auto th = co::thread([]() -> co::func<void>
{
    co_await co::this_thread::sleep_for(100ms);
});
co_await th.join();
co::thread([]() -> co::func<void>
{
    co_await co::this_thread::sleep_for(100ms);
}).detached();
